# Customer Churn Analysis

## Project Overview
This project analyzes telecom customer churn data to identify factors affecting customer retention.

## Tasks Performed
- Data manipulation using pandas
- Data visualization using matplotlib and seaborn
- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest

## Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

## Objective
To analyze telecom customer data and identify patterns leading to customer churn.
